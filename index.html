자유 저장소

사용자명, 검색어, URL로 어떤 GitHub 저장소도 문제로 사용할 수 있습니다.

자동 채점

Pyodide로 브라우저에서 Python을 직접 실행하여 정답을 자동 비교합니다.

Google Sheet 연동

Apps Script URL만 입력하면 모든 제출 기록이 자동 저장됩니다.

사용법

  1. 학습 탭 → GitHub 저장소 검색 또는 직접 입력 → 문제 로드
  2. 문제 선택 → 학습 시작 버튼 클릭 → 코드 작성 → 제출
  3. 관리 탭에서 Apps Script URL 입력하면 기록이 Google Sheet에 자동 저장
