자동 채점

Pyodide로 브라우저에서 직접 Python 실행. 정답 코드와 출력을 비교해 자동 채점합니다.

Google Sheet 연동

Apps Script URL만 입력하면 모든 제출 기록이 Google Sheet에 자동 저장됩니다.

사용법 (3단계)

  1. 학습 탭 → GitHub 저장소 검색 또는 직접 입력 → 문제 로드
  2. 문제 선택 → 코드 작성 → 실행 / 제출
  3. 관리 탭에서 Apps Script URL 입력하면 기록이 Google Sheet에 자동 저장됨
