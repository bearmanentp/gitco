자유 모드

사용자가 GitHub 사용자명 / 저장소명 / URL을 직접 넣어 어떤 공개·비공개 레포도 문제은행처럼 사용할 수 있습니다.

기록 분리

학생은 본인 기록만 보고, 선생님은 전체/반별 기록을 관리합니다.

라이브러리 지원

repo 안의 libraries 폴더를 자동 로드하여 Pyodide에서 지원이 애매한 순수 파이썬 라이브러리를 불러올 수 있습니다.

빠른 시작

  1. Google Sheet에 Apps Script를 붙여넣고 웹앱으로 배포
  2. GitHub에 문제 .py 파일 업로드
  3. 학습 탭에서 repo 검색/선택 후 문제 불러오기
